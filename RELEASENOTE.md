# Release note v1.6.7: 合計値の確認

## 機能修正
- **Excel読込修正**: `excelize.Options{RawCellValue:true}` を使用し、表示形式に左右されない正確な値を取得 ※日付セルはExcelシリアル値として安全に読み込み後、独自でパース
- **合計検証強化**:
  - シートを走査して全シートの合計値を自動検証
  - `validateExcelSums()` を型・関数に分割し保守性向上
  - シート名変更・注文数変動にも対応（印刷シート名変更時も自動判定）
- **軽微な改善**: 入力IIの合計チェックを固定値化、不要なmagic定数除去
